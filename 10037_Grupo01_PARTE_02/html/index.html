Animaciones DOM

JavaScript HTML DOM Animación se refiere a la capacidad de usar JavaScript para crear animaciones interactivas en páginas web mediante la manipulación del Modelo de Objetos del Documento (DOM) HTML. La animación en JavaScript se logra mediante la actualización continua de las propiedades de estilo de un elemento HTML en un bucle de animación. Por ejemplo, puedes cambiar la posición, tamaño, opacidad, color u otras propiedades de un elemento para crear efectos visuales como desvanecimientos, desplazamientos, transiciones y transformaciones. Existen varias formas de crear animaciones en JavaScript utilizando el DOM. Una forma común es utilizar el método `setInterval()` para ejecutar una función repetidamente en intervalos regulares. Dentro de esa función, puedes actualizar las propiedades de estilo de un elemento para lograr la animación deseada. Además, existen bibliotecas y frameworks de JavaScript, como jQuery y GSAP (GreenSock Animation Platform), que facilitan la creación de animaciones más complejas y eficientes. Estas bibliotecas proporcionan métodos y funcionalidades adicionales para simplificar el proceso de animación y optimizar el rendimiento.

Ejemplo

container.animate({
                transform: [
                    'rotateX(60deg) rotateZ(0deg)',
                    'rotateX(60deg) rotateZ(360deg)',
                ],
                    }, {
                        duration: 20000,
                        iterations: Infinity,
                });
                for (var y = -7; y <= 7; y++) {
                for (var x = -7; x <= 7; x++) {
                    var box = createBox();
                    box.animate({
                    transform: [
                        'translateZ(0px)',
                        'translateZ(20px)',
                    ],
                    opacity: [1, 0],
                    }, {
                        delay: (x*x + y*y) * 20,
                        duration: 2000,
                        iterations: Infinity,
                        direction: 'alternate',
                        easing: 'ease-in',
                    });
                    }
                }
                function createBox() {
                    var box = document.createElement('div');
                    box.className = 'box';
                    container.appendChild(box);
                    return box;
                }


DOM Eventos

JavaScript HTML DOM Events se refiere a la capacidad de JavaScript de detectar y responder a eventos que ocurren en los elementos del Document Object Model (DOM) de una página HTML. Existen muchos tipos de eventos en JavaScript, entre los más comunes se encuentran: - Eventos de ratón: como "click" (clic), "mouseover" (pasar el ratón por encima), "mouseout" (sacar el ratón de encima), entre otros. - Eventos de teclado: como "keydown" (tecla presionada), "keyup" (tecla liberada), "keypress" (tecla presionada y liberada), entre otros. - Eventos de formulario: como "submit" (enviar formulario), "input" (cambio en un campo de entrada), "focus" (enfoque en un elemento), entre otros. - Eventos de carga: como "load" (carga completa de la página), "DOMContentLoaded" (cuando el DOM está listo), entre otros. Para manejar los eventos, se pueden utilizar métodos como `addEventListener` para asignar una función a un elemento y especificar qué evento se debe escuchar. Cuando ocurre ese evento, la función se ejecuta y puede realizar acciones específicas, como cambiar el contenido de la página, realizar cálculos, hacer solicitudes al servidor, etc.

Ejemplo

DOM Event Listener

DOM Navegacion

La navegación en JavaScript HTML DOM se refiere a la capacidad de acceder y manipular los elementos del Document Object Model (DOM) utilizando diferentes métodos de navegación. El DOM representa la estructura de un documento HTML como un árbol de nodos, donde cada elemento, atributo y texto se representa como un nodo en el árbol. JavaScript proporciona una serie de métodos de navegación que te permiten moverte a través de los nodos del DOM y acceder a elementos específicos o buscar elementos que cumplan ciertos criterios. Algunos de los métodos de navegación más comunes son: -getElementById(id): Retorna el elemento con el atributo "id" especificado. -getElementsByClassName(className): Retorna una colección de elementos que tienen el nombre de clase especificado. -getElementsByTagName(tagName): Retorna una colección de elementos que tienen el nombre de etiqueta especificado. -querySelector(selector): Retorna el primer elemento que coincide con el selector CSS especificado. -querySelectorAll(selector): Retorna todos los elementos que coinciden con el selector CSS especificado. Estos métodos de navegación te permiten acceder a elementos individuales o a colecciones de elementos que cumplen ciertos criterios. Puedes usar métodos de navegación para realizar tareas como seleccionar elementos para modificar su contenido, agregar o eliminar elementos del DOM, cambiar estilos, entre otros.

Ejemplo

var navegacion = function(){
                                document.getElementById("f2").innerHTML = document.getElementById("fila4").innerHTML;
                                document.getElementById("f3").innerHTML = document.getElementById("fila2").innerHTML;
                                document.getElementById("f4").innerHTML = document.getElementById("fila3").innerHTML;
                                }

DOM Nodos

En el contexto del DOM, un nodo es un único punto en el arbol de nodos. Los nodos pueden ser varias cosas el documento mismo, elementos, texto y comentarios. Los principales tipos de nodos en el DOM son:

a. Document: el nodo document es el nodo raíz, a partir del cual derivan el resto de nodos.

b. Element: son los nodos definidos por etiquetas html. Por ejemplo una etiqueta div genera un nodo. Si dentro de ese div tenemos tres etiquetas p, dichas etiquetas definen nodos hijos de la etiqueta div.

c. Text: el texto dentro de un nodo element se considera un nuevo nodo hijo de tipo text (texto). Los navegadores también crean nodos tipo texto sin contenido para representar elementos como saltos de línea o espacios vacíos.

d. Attribute: los atributos de las etiquetas definen nodos, aunque trabajando con JavaScript no los veremos como nodos, sino que lo consideramos información asociada al nodo de tipo element.

e. Comentarios y otros: los comentarios y otros elementos como las declaraciones doctype en cabecera de los documentos HTML generan nodos.

Para agregar un nuevo elemento al HTML DOM, primero debe crear el elemento (nodo de elemento) y luego agregarlo a un elemento existente.

Ejemplo

var agregarparrafo = function(){
                            var min=1;
                            var max=6;
                            
                            const para = document.createElement("p");
                            const node = document.createTextNode(Math.floor(Math.random()*(max-min+1)+min));
                            para.appendChild(node);
                            const element = document.getElementById("div1");
                            element.appendChild(para);
                            }

  • HTMLCollection.namedItem()

    • Devuelve el nodo específico cuyo ID coincide con la cadena especificada por nombre.

    DOM Lista de Nodos

    En JavaScript, una lista de nodos se refiere a una colección de elementos HTML que se almacenan en un objeto especial llamado NodeList. Un NodeList es similar a un array (matriz) en JavaScript, pero en lugar de contener métodos y propiedades específicos de las matrices, proporciona métodos y propiedades específicos para trabajar con nodos HTML. Cuando se realiza una consulta o selección de elementos en el Document Object Model (DOM) utilizando métodos como 'querySelectorAll()' o 'getElementsByTagName()', obtienes un NodeList que contiene los nodos coincidentes. Es importante tener en cuenta que un NodeList es una "lista viva" y reflejará cualquier cambio en el DOM. Esto significa que si se agrega o elimina un elemento del DOM después de obtener un NodeList, el NodeList se actualizará automáticamente para reflejar esos cambios. 

    var nodeList = function () {
                        const nodeList = document.querySelectorAll("input");
                        for (let i = 0; i < nodeList.length; i++) {
                        nodeList[i].style.background = "#e474f1";
                        nodeList[i].style.color = "green";
                        }
                    }
